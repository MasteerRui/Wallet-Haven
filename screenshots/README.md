# WalletHaven Screenshots

This folder contains additional screenshots of the WalletHaven application showcasing various features and screens.

## Screenshot Gallery

Add your screenshots here to showcase:

- **Home Screen**: Main dashboard with wallet overview and recent transactions
- **Transactions**: Complete transaction history with filtering and search
- **Goals**: Financial goals tracking with progress visualization
- **Settings**: User preferences, security settings, and account management
- **Wallet Management**: Create, edit, and manage multiple wallets
- **AI Receipt Scanning**: Camera interface for scanning receipts and invoices
- **Category Management**: View and manage expense categories
- **Dashboard & Analytics**: Charts and statistics for financial insights
- **Authentication**: Sign in, sign up, and password reset screens

## Adding Screenshots

When adding screenshots:

1. Use descriptive filenames (e.g., `home-screen.png`, `transaction-list.png`)
2. Optimize images for web (recommended: PNG or JPG, max 2MB per image)
3. Include both iOS and Android screenshots if available
4. Update this README with descriptions of new screenshots

## Screenshot Guidelines

- **Resolution**: Minimum 1080x1920 pixels (or equivalent aspect ratio)
- **Format**: PNG or JPG
- **Naming**: Use kebab-case (e.g., `ai-receipt-scan.png`)
- **Privacy**: Ensure no sensitive personal information is visible in screenshots
